## 🗨️ About SSAM
추락하는 교권을 보호하기 위한 상담 시스템. **SSAM**<br>
학부모의 끊임없는 연락, 폭언을 막기 위해 기획한 서비스입니다.  
SSAM에서 제공하는 주요 기능은 다음과 같습니다.
1. 학급 및 학생 관리
2. 화상 상담 및 실시간 욕설 필터링 기능
3. 학급 공지 챗봇 관리
